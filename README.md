## ℹ️ About
Automatically downlaods and applies a backup of Bazarr from an sftp when starting up.

## 🏃‍➡️ How to run
Except from the standard setup you would use for Bazarr, you also need the following environment variables:
 - SFTP_SERVER
 - SFTP_USERNAME
 - SFTP_PASSWORD
